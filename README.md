# 🤖 Spam Email Detection System

### Machine Learning-Based Text Classification Project

---

## 📌 Project Overview
This project builds a machine learning model to classify messages as spam or not spam.

---

## 📂 Dataset Used
**SMS Spam Dataset**
- Message text  
- Label (spam / ham)

---

## 🛠️ Tools & Technologies
- Python  
- Pandas  
- Scikit-learn  
- TF-IDF Vectorizer  
- Naive Bayes  

---

## 🔄 Project Workflow
1. Data preprocessing  
2. Label encoding  
3. Train-test split  
4. Text vectorization  
5. Model training  
6. Prediction  

---

## 📈 Key Insights
- High accuracy (~90–97%)  
- Spam messages contain strong keyword patterns  
- Text classification is effective for filtering

- Spam detection model  
- Accuracy evaluation
